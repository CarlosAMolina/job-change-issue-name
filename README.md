# job-modify-issue-name

## Introduction

Modify the name of a Jira's issuer to create git branches, folders, etc.

## Test

Modify the `src/modify-name.js` file:

- Comment the lines with `document`.
- Uncomment the `exportedForTesting` constant.

Run:

```bash
npm install
npm test
```
